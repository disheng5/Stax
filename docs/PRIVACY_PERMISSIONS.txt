Stax 小程序 - 用户隐私保护指引（填表参考）

用于微信小程序后台「设置 → 服务内容声明 → 用户隐私保护指引」配置。

================================================================

需要在后台开启 / 声明的信息

----------------------------------------------------------------
1. 用户身份相关
----------------------------------------------------------------

【openid】
  - 用途：标识用户身份，关联战绩与牌局参与记录
  - 调用接口：wx.cloud.callFunction('whoami')
  - 是否传至第三方：否（仅存储于开发者授权的腾讯云开发环境）

【微信昵称（nickname）】
  - 用途：在牌局中向其他玩家展示
  - 调用接口：用户在「我的」页面通过 input type="nickname" 主动输入
  - 是否传至第三方：否

【微信头像（avatarUrl）】
  - 用途：在牌局中向其他玩家展示
  - 调用接口：button open-type="chooseAvatar"
  - 是否传至第三方：否

----------------------------------------------------------------
2. 设备能力
----------------------------------------------------------------

【相册写入权限（writePhotosAlbum）】
  - 用途：将牌局结算图保存到用户本地相册，便于分享
  - 调用接口：wx.saveImageToPhotosAlbum
  - 触发场景：结算页用户主动点击「保存相册」按钮
  - 是否传至第三方：否

【摄像头 / 扫码（scope.camera）】
  - 用途：扫描朋友的牌局邀请二维码以加入牌局
  - 调用接口：wx.scanCode
  - 触发场景：加入牌局页用户主动点击「扫码加入」按钮
  - 是否传至第三方：否

----------------------------------------------------------------
3. 剪贴板
----------------------------------------------------------------

【写入剪贴板（setClipboardData）】
  - 用途：
    a. 复制 6 位牌局邀请码
    b. 复制 AI 点评文本 / 术语解释文本
  - 调用接口：wx.setClipboardData
  - 触发场景：用户主动点击「复制」按钮
  - 是否传至第三方：否

================================================================

不需要开启 / 声明的项（小程序未调用，请勿勾选）

  - 手机号（getPhoneNumber）
  - 地理位置（getLocation / chooseLocation）
  - 通讯录（chooseContact）
  - 微信运动步数
  - 录音
  - 蓝牙 / NFC / WIFI
  - 读取剪贴板（getClipboardData）
  - 相册读取 / 选图（chooseImage / chooseMedia）
  - 摄像头录像（不上传，仅扫码用）

================================================================

数据存储与共享

  - 所有数据存储于腾讯云开发（CloudBase）由开发者授权的环境内
  - 不向任何第三方传输或共享
  - 用户分享牌局时仅传递昵称、头像、邀请码至接收方（同样为微信用户）

================================================================

用户权利

  - 在「我的」可清除本地缓存
  - 在「战绩」页可随时删除自己的单条牌局记录（不影响其他玩家）
  - 通过 GitHub Issue 联系开发者申请彻底删除云端账户
