# Stax 隐私协议

发布日期：2026-05-12

感谢使用 Stax · 长河筹略（以下简称"本小程序"）。本协议说明我们在你使用本小程序时收集、使用、存储与共享的信息，以及你享有的权利。

## 1. 我们收集的信息

| 类型 | 用于 | 何时收集 |
|---|---|---|
| 微信 openid | 唯一标识用户、关联战绩 | 首次进入小程序 |
| 微信昵称 / 头像 | 在牌局中向其他玩家展示 | 你点击「授权登录」后 |
| 牌局数据（局名 / 买入额 / 盲注 / 玩家筹码 / 流水） | 提供记账、计时、统计功能 | 创建/加入牌局并进行操作时 |

我们 **不收集** 真实姓名、手机号、地址、相册、定位等敏感信息。

## 2. 我们如何存储

所有数据保存在腾讯云开发（CloudBase）你授权的环境内，不向任何第三方传输。

## 3. 共享与披露

除以下情形外，我们不向任何第三方共享你的信息：
- 法律法规明确要求；
- 你主动通过分享卡片邀请其他用户加入你的牌局（仅传递昵称头像与牌局邀请码）。

## 4. 你的权利

- 在「我的」→「清除缓存」可清除本地用户信息；
- 通过 GitHub Issue / 客服邮箱（待补充）可申请彻底删除云端账户。

## 5. 安全声明

本小程序仅用于**朋友间线下竞技扑克记账与学习交流**，**严禁用于任何形式的赌博活动**。请遵守当地法律法规。

## 6. 协议更新

本协议如有更新会在本页面公示，重大变更将通过小程序内弹窗通知。

## 7. 联系我们

通过 GitHub 仓库 Issue 区域提交反馈。
